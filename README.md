# Discord HypeSquad Badge Claimer

A lightweight and experimental tool for claiming a Discord HypeSquad House badge through Discord's available API endpoint.

> ⚠️ **Disclaimer**
> This project is provided **strictly for educational, research, and experimental purposes**.
> The authors and contributors **do not encourage or endorse violating Discord's Terms of Service** and accept **no responsibility** for any misuse, account restrictions, suspensions, or damages resulting from the use of this software.

---

## ✨ Features

- 🎖️ Claim a HypeSquad House badge
- ⚡ Fast and lightweight
- 🖥️ Simple command-line interface
- 📦 Minimal dependencies
- 🔧 Easy configuration

---

## 📥 Installation

Clone the repository:

```bash
git clone https://github.com/jdvnn/discord-hypesquad-badge-claimer.git
cd discord-hypesquad-badge-claimer
````

---

## 🚀 Usage

Run the application:

```bash
python claimer.py
```

Follow the on-screen instructions.
